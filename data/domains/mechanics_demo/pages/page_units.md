# SI units in mechanics

Mechanics calculations stay sane when quantities use SI base units.

- **Mass** is measured in **kilogram** (kg).
- **Force** is measured in **newton** (N).
- By definition, **1 N = 1 kg·m/s²**.

Always check dimensions before you trust a numeric answer.
